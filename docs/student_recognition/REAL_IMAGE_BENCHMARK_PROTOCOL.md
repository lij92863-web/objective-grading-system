# Anonymous Real Image Benchmark Protocol

Local material belongs only in `local-test-materials/student_recognition_real_images/`, which is gitignored. Cards must be blank or self-created, anonymous, contain no real student identity or answers, and have `ground_truth.json` with case id, relative image, SHA-256 and `contains_real_student_data: false`. Missing data reports NOT_RUN. Hash/privacy mismatch reports FAIL. Metrics remain null until an approved local run; no synthetic result may be presented as real-image evidence.
